HTML SELECTOR CHEASTSHEET

Basic Selectors

Selector Example Description
elementName p Selects all <p> elements
.class .intro select elements with class ="intro"
#id #firstname Selects the element with id="firstname"
[attr=value] [target=_blank] Selects all elements with target="_blank"
* * selects all HTML elements on the page

Combinators

Selector Example Description
selectorA + selectorB div + p Selects the first <p> element that are placed immediately after
elements
selectorA ~ selectorB p ~ ul Selects every <ul> element that are preceded by a <p> element
parent > child div > p Selects all <p> elements where the parent is a <div> element
parent descendent div p Selects all <p> elements inside <div> elements

Pseudo Selectors

Selector Example Description
:active a:active Selects the active link
:hover a:hover Selects links on mouse over
:visited a:visited Selects all visited links
:focus input:focus Selects the <input> element that has focus
